# BlankFX Docs

BlankFX is the stablecoin FX infrastructure layer for businesses that need fast settlement, tight pricing, and a clean audit trail.

## What it does

BlankFX is built for corridor-based settlement rather than generic token swapping. The docs cover pricing, routing, liquidity, compliance, and contracts.
